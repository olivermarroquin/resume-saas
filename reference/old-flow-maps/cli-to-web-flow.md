# CLI to Web Flow Map

## Old CLI Flow

user
→ terminal command
→ local script
→ local files / outputs
→ manual review

## New Web Flow

user
→ web UI
→ backend API
→ backend services
→ structured response
→ UI display / download

## Translation Rule

Replace command-driven actions with UI/API-driven actions.
Replace local file-state assumptions with app state and service calls.
