# Resume SaaS Overview

## Goal

Build a web application that allows users to generate tailored resumes through a UI instead of terminal commands.

## Source Logic

The source logic comes from Resume Factory.

Key reusable areas identified so far:

- resume parsing
- JD term extraction
- proposal validation
- rewrite packet formatting

## App Shape

Resume SaaS will be organized around:

- backend services
- backend APIs
- frontend UI
- reference mapping to source logic

## Rule

The new app should be built from extracted engine logic, not from copying local workflow clutter.
